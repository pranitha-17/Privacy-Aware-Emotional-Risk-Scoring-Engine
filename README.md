# Privacy-Aware-Emotional-Risk-Scoring-Engine
Most mental health apps wait for you to ask for help — by then it's often too late. This system silently detects emotional risk from behavioral signals (screen time, social withdrawal, activity drops) and flags it early, with everything processed on-device, so zero private data ever leaves the phone.

Behavioral signals like app usage patterns, screen time, social withdrawal, and activity drops can quietly indicate that someone is struggling mentally — long before they reach out for help. But existing mental health tools either require users to self-report (which people avoid due to stigma) or send sensitive data to the cloud (which kills privacy and trust).

This system detects emotional risk silently and privately — all inference runs on-device, raw behavioral data never leaves the phone, and only an anonymized risk tier (low/medium/high) is output — along with a gentle, non-clinical nudge toward the right resources.
